When you add resource more resource, you can modify the resource.xml.
1.If you want to add some resource under a existing section, you can just add it under that section.
2.If you want to add a new section, you can use:
<section>
	<section_title></section_title>
        <link>
		<title></title>
		<url></url>
	</link>
</section>

The resource.xml is easy to understand.


if you want to change the format, please modify resource.php.

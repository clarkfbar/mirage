if you want to add new people, modify people.xml
if you want to change the format, modify people.php

name of the image in photo must be in this format: "pic-(first letter of first name)(last name).jpg"    i.e.  Zeyu Feng => pic-zfeng.jpg

To modify the people.xml:
There are four sections: faculty, visitor, cstudent(current student),pstudent(past student)
Please add the people under the correct section. If they have no photo, just leave it blank
